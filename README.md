# quantx-lab

jupyterでQuantXアルゴリズムを開発！
QuantX-SDK を使用したnotebook実行環境です。

## 準備
- docker
- docker-compose

## 使い方

```
docker-compose up lab
```

もしくは

```
docker run -p 8888:8888 smartradev/quantx-lab
```

jupyter labが起動するので、コンソールに表示されたURLをブラウザに入力してアクセスしてください。
